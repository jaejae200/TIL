### 배운 점

조금 오래 걸리더라도 내 능력으로 풀어나갈 수 있는 문제를 늘려나가고 있다.

아직 모듈 사용이 익숙하지 않은 것 같다.

아직 그래도 자료나 구글링에 많이 의존하게 되는 것 같다.
